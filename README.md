# 现代俄罗斯方块游戏

一个使用纯HTML、CSS和JavaScript实现的现代化俄罗斯方块游戏。

## 游戏特点

- 🎮 经典玩法：包含7种不同形状的方块，通过旋转和移动来消除行
- 👁️ 方块预览：显示下一个将出现的方块，帮助玩家提前规划
- 👻 幽灵方块：显示当前方块落地位置的半透明提示
- 🔢 分数系统：根据消除行数计算得分，并显示当前等级
- 🎚️ 难度选择：提供初级、中级、高级三种难度选择
- 🎛️ 游戏控制：支持暂停/继续、重新开始功能

## 操作说明

- ← → : 左右移动方块
- ↑ : 旋转方块
- ↓ : 加速方块下落
- 空格 : 方块直接落到底部
- P : 暂停/继续游戏
- R : 重新开始游戏
